# git-practice

이프로젝트는 git을 연습하기 위한 저장소입니다.

See [Demo](https://www.google.com/)

## Documentation

### Installation

```shell
$ git clone {repo url}
$ cd {reponame}
$ pip install -r requirements.txt
```

### How To Start

```shell
$ python main.py
```

### Dependancy

- Python==3.10
- Django==3.0.0
- requests^0.10.0

- save random numbers
